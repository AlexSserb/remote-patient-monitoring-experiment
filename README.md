# Remote Patient Monitoring — Experiment

Сравнительный эксперимент алгоритмов умных уведомлений для системы удалённого мониторинга пациентов.

## Цель

Выбрать наилучший алгоритм выбора времени отправки уведомлений, максимизирующий регулярность заполнения дневников здоровья пациентами и опекунами.

## Роли в системе

| Роль | Кол-во | Описание |
|------|--------|----------|
| Пациент | 1000   | Заполняет дневник самостоятельно |
| Опекун | 200    | Заполняет дневник за пациента |
| Врач | -      | Не участвует в эксперименте |

## Алгоритмы

### 1. Статический
Время уведомления фиксируется один раз - по субъективному предпочтению пользователя, указанному при регистрации. Не адаптируется в течение эксперимента.

### 2. Эмпирический
Стартует с субъективного лучшего времени. Адаптируется по правилам:

1. Скользящее окно 7 дней: если средняя награда < 0.3 -> сдвиг на +1 временной слот.
2. Если средняя награда >= 0.5 -> время зафиксировано.
3. Направление сдвига - hill-climbing: сначала +1 слот, если через 7 дней хуже - меняет на −1.
4. Каждые 30 дней - принудительная переоценка: выбирается слот с наилучшей наградой за последние 30 дней.

### 3. D-UCB (Discounted UCB)

Multi-armed bandit с дисконтированием прошлых наблюдений.

- **Автоматы:** 14 временных слотов по 1 часу, с 08:00 до 22:00.
  Уведомление отправляется в начале слота: 08:00, 09:00, ..., 21:00.
- **Параметр дисконтирования:** γ = 0.97 (период полураспада ≈ 23 дня).
  Обеспечивает адаптацию к изменениям поведения в течение нескольких недель без полного забывания истории.
- **Формула выбора слота:**
  ```
  UCB_i(t) = Q̂_i(t) + c * sqrt(ln(N̂(t)) / n̂_i(t))
  ```
  где `Q̂_i`, `n̂_i`, `N̂` — дисконтированные суммы наград, числа выборов и общего числа шагов.
- **Инициализация:** оптимистичная — каждый слот посещается по одному разу перед основным циклом.

## Функция награды

```
R = 0.2 * O + 0.3 * D + 0.5 * F
```

| Переменная | Тип | Описание |
|-----------|-----|---------|
| O | {0, 1} | Открыл ли пользователь систему |
| D | {0, 1} | Заполнены ли все **обязательные** поля дневника |
| F | [0, 1] | Доля заполненных полей от всех возможных |

**Инварианты:** D=0 => F=0; без заполнения обязательных полей форма не отправляется.

## Метрики эксперимента

Все метрики считаются накопительно за 180 дней.

| Метрика | Формула | Описание |
|---------|---------|---------|
| **R** | среднее за прогон | Средняя функция награды |
| **OSR** | ∑O / 180 | Open/Session Rate |
| **DCR** | ∑D / 180 | Diary Completion Rate |
| **AFFR** | ∑(заполн. полей) / ∑(всех полей) | Average Field Fill Rate |
| **ART** | среднее времени реакции | Без учёта игнорирований |

## Симуляция пользователей

### Профиль пользователя

| Параметр | Тип | Описание |
|---------|-----|---------|
| `subjective_best_time` | float[7] | Субъективно лучшее время по дням недели (часы) |
| `objective_best_time` | float[7] | Объективно лучшее время по дням недели (часы) |
| `weekday_coefficients` | float[7] | Мультипликаторы вероятности реакции по дням недели |
| `adherence` | float | Базовая вероятность взаимодействия (высокая/средняя/низкая) |
| `reaction_speed` | LogNormal | Параметры распределения задержки реакции; пик скорости совпадает с objective_best_time |
| `diagnoses` | list[Diagnosis] | Диагнозы, определяющие поля дневника |
| `seed` | int | Seed для воспроизводимости |

Вероятность просмотра уведомления максимальна в момент объективно лучшего времени и убывает по гауссову закону при отклонении от него.

## Структура эксперимента

- **Длительность:** 180 дней, 1 уведомление в день
- **Прогонов:** 10 независимых для каждого алгоритма
- **Пользователи:** фиксированы по seed (профили одинаковы для всех прогонов одного алгоритма)
- **Сравнение:** среднее по 10 прогонам для каждой метрики

## Диагнозы

Описаны в `data/diagnoses.py`. Используется 5 диагнозов, каждый определяет набор обязательных и необязательных полей дневника. Пользователю может быть назначен один или несколько диагнозов.

| Диагноз | Обязательных полей | Необязательных полей |
|---------|-------------------|---------------------|
| Гипертония | 3 | 3 |
| Сахарный диабет 2 типа | 4 | 3 |
| Хроническая сердечная недостаточность | 4 | 3 |
| ХОБЛ | 3 | 3 |
| Остеоартрит | 3 | 3 |
